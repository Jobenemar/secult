Instruções de hospedagem

Para uso no servidor da Prefeitura

1. GitHub Pages (gratuito e recomendado):
Crie um repositório privado no GitHub, suba os dois arquivos HTML, ative o GitHub Pages nas configurações. O formulário público fica em seuusuario.github.io/repo/pmc_formulario_publico.html e o painel em seuusuario.github.io/repo/pmc_admin.html. Por serem do mesmo domínio, o storage é compartilhado.

2. Servidor da Prefeitura / hospedagem municipal:
Suba os dois arquivos na mesma pasta do servidor via FTP ou painel de hospedagem. O storage funciona por localStorage do navegador partilhado entre arquivos do mesmo domínio.

3. Alterar senhas de acesso:
Abra o arquivo pmc_admin.html em um editor de texto, localize o objeto USERS e altere os valores de pass para as senhas desejadas.
